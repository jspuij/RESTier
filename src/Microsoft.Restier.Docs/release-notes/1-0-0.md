### Features
- ASP.NET Classic 4.7.2 and later support
  - Supports Entity Framework Classic 6.x
- ASP.NET Core 3.1 and later support
  - Supports Entity Framework Classic 6.x
  - Supports Entity Framework Core 5.0 and later
